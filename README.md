# Awesome Hello Languages [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> One repository. One greeting. 214 programming languages and data formats.

A curated, runnable collection of **"Hello, World!"** — the same line printed in
as many different languages as possible. Compiled, interpreted, functional,
logic, stack-based, esoteric, hardware-description, query and markup languages
all live side by side under [`languages/`](languages).

Every file is generated by [`generate.py`](generate.py), so the catalog is a
single source of truth: add an entry to the map, re-run the script, and the
greeting appears in a new tongue.

## How it works

The whole repo is driven by one catalog in [`generate.py`](generate.py):

```python
PROGRAMS = {
    "Rust": ("rust/hello.rs", 'fn main() { println!("Hello, World!"); }'),
    # ...
}
```

Run it to (re)materialize every file:

```bash
python generate.py        # writes languages/**/hello.*
python build_readme.py    # regenerates this README
```

## Run a few

```bash
python  languages/python/hello.py
ruby    languages/ruby/hello.rb
node    languages/javascript/hello.js
go      run languages/go/hello.go
bash    languages/bash/hello.sh
lua     languages/lua/hello.lua
```

## Contributing

1. Add a `"Language": ("dir/hello.ext", "source")` entry to `PROGRAMS`
   (or `MARKUP`) in `generate.py`.
2. Run `python generate.py && python build_readme.py`.
3. Open a pull request. Keep the output exactly `Hello, World!`.
